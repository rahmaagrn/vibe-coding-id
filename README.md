**Panduan vibe coding BAHASA INDONESIA pertama untuk pemula total.**

## Apa Ini?

Vibe Coding Starter Kit adalah **panduan lengkap bahasa Indonesia** untuk orang yang ingin belajar bikin software dengan AI — tanpa perlu pengalaman coding sebelumnya.

**Penjelasan di luar sana masih menakutkan untuk pemula.**
Proyek ini ada untuk mengubah itu.

## Untuk Siapa?

- 🎯 **Orang Indonesia** yang belum pernah coding
- 🎯 **Pemula total** — tidak perlu tahu apa itu "syntax" atau "framework"
- 🎯 **Vibe coder baru** — sudah dengar soal vibe coding tapi bingung mulai dari mana
- 🎯 **Siapa pun** yang ingin bikin website, aplikasi, atau tool tanpa belajar programming

## Fitur

| Fitur | Detail |
|-------|--------|
| 🌐 Bahasa 100% Indonesia | Tidak ada lagi pusing membaca tutorial bahasa Inggris |
| 📖 3 Tingkatan Belajar | Dari Nol Koding → Mulai Bikin → Mulai Paham |
| 🔍 Glossary Interaktif | Hover/tap pada istilah teknis untuk lihat penjelasan |
| 🌗 Dark mode | Nyaman dibaca kapan saja |
| 📱 Mobile-first | Dioptimasi untuk pengguna HP |
| 🔎 Search built-in | Cari konten dengan cepat |
| ⚡ Fast | < 200KB per halaman, load di 3G < 3 detik |

## Mulai Cepat

```bash
# Clone
git clone https://github.com/rahmaagrn/vibe-coding-id.git

# Install dependencies
cd vibe-coding-id
npm install

# Development server
npm run dev

# Build untuk production
npm run build
```

Atau langsung kunjungi **https://rahmaagrn.github.io/vibe-coding-id/** (sedang di-deploy).
