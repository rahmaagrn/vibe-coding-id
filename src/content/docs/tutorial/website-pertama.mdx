Ini adalah halaman tutorial. Ke depannya, konten tutorial dari materi-materi yang sudah kita buat akan ditempatkan di sini.

## Yang Kamu Butuhkan

- **Tool AI coding** pilihanmu (lihat [Pilih Tool](/vibe-coding-id/mulai/pilih-tool/) kalau belum punya)
- **Koneksi internet**
- **10 menit waktu luang**

## Langkah 1: Buka Tool AI Coding-mu

Buka Claude Code, Cursor, Bolt.new, atau Replit. Pilih satu yang paling nyaman.

## Langkah 2: Copy-Paste Prompt Ini

Salin prompt di bawah ini dan tempel ke tool AI coding kamu:

```
Saya ingin membuat sebuah website pribadi (personal landing page) 
dengan spesifikasi berikut:

- Nama saya: [nama kamu]
- Background warna gradien dari biru ke ungu
- Tampilan modern, minimalis, dan responsive (bisa dibuka di HP)
- Ada bagian: Hero (nama + tagline singkat), Tentang Saya, 
  Skill (3-4 skill icon), Kontak (tombol email)
- Font yang bersih dan mudah dibaca
- Satu file HTML saja, tanpa framework

Tolong buatkan kode lengkapnya dengan CSS internal.
```

## Langkah 3: Lihat Hasilnya

Tool akan mengeluarkan kode HTML. Kalau kamu pakai Bolt.new atau Replit, hasilnya langsung keliatan. Kalau Claude Code atau Cursor, simpan file sebagai `index.html` dan buka di browser.

**Selamat!** Kamu baru saja bikin website pertamamu dengan vibe coding. 🎉

## Langkah 4: Iterasi

Sekarang minta AI untuk mengubah sesuatu:

- *"Tambah foto profile di bagian hero"*
- *"Ganti background jadi warna hijau"*
- *"Tambah footer dengan ikon media sosial"*
- *"Buat animasi halus saat scroll"*

Setiap perubahan adalah proses **iterasi** — dan ini inti dari vibe coding.
