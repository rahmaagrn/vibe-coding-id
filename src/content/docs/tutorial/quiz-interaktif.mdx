Bikin kuis interaktif adalah proyek vibe coding yang sempurna untuk pemula. Hasilnya seru, bisa langsung dipakai, dan gampang di-custom.

## Yang Kamu Butuhkan

- Tool AI coding pilihanmu
- 15 menit waktu luang
- Ide topik kuis (misal: "Seberapa Indonesia Kamu?" atau "Quiz Pengetahuan Umum")

## Prompt untuk AI

Copy-paste prompt ini ke tool AI coding kamu:

```
Buat sebuah website kuis interaktif satu halaman (single HTML file) dengan:

- Judul kuis: "[topik kuis kamu]"
- 5 pertanyaan pilihan ganda, masing-masing 4 opsi
- Setiap jawaban benar: +20 poin
- Setelah selesai, tampilkan skor akhir dengan pesan personalisasi:
  - 100: "Quiz Master! 🏆"
  - 80: "Hebat! 🎉"
  - 60: "Cukup bagus! 👍"
  - <60: "Ayo coba lagi! 💪"
- Tampilan colorful, playful, dan responsive (bisa di HP)
- Animasi saat ganti pertanyaan
- Progress bar yang menunjukkan pertanyaan ke berapa
- CSS internal, satu file HTML saja

Gunakan emoji dan warna yang ceria. Buat semenarik mungkin!
```

## Kustomisasi

Setelah kuis pertama jadi, coba modifikasi:

- *"Tambah timer 15 detik per pertanyaan"*
- *"Tambah sound effect saat jawab benar/salah"*
- *"Buat leaderboard lokal pakai localStorage"*
- *"Ganti tema jadi horror untuk quiz malam minggu"*
- *"Tambah 5 pertanyaan lagi"*
