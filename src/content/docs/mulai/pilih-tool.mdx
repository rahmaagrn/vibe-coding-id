## Perbandingan Cepat

| Tool | Yang Paling Cocok Untuk | Butuh Install? | Harga Mulai |
|------|------------------------|----------------|-------------|

## Claude Code — Untuk Kamu yang Suka Ngobrol dengan AI

Claude Code adalah tool dari Anthropic (perusahaan yang bikin Claude AI). Cara kerjanya: kamu ngobrol dengan AI di terminal, dan AI-nya yang nulis dan menjalankan kode.

**Kelebihan:**
- Paling powerful untuk proyek yang kompleks
- Bisa akses file langsung di komputermu
- Bisa nulis, edit, dan running kode — semua dari chat
- Cocok untuk proyek serius yang skalanya besar

**Kekurangan:**
- Perlu install dan setup terminal — sedikit lebih teknis
- Antarmuka cuma teks (terminal), tidak ada tampilan drag-and-drop
- Akses Claude Code Max butuh GitHub project impactful (gratis) atau berbayar

**Coba sekarang:** Buka [claude.ai/code](https://claude.ai/code)

## Cursor — Seperti VS Code Tapi dengan AI

Cursor adalah editor kode (seperti VS Code) yang sudah dilengkapi AI di dalamnya. Kamu ngetik seperti biasa, tapi AI bisa bantu nulis, memperbaiki, dan menjelaskan kode.

**Kelebihan:**
- Tampilan familiar (seperti VS Code) — lebih ramah untuk pemula
- AI bisa suggest kode saat kamu ngetik
- Bisa tanya AI tentang kode yang sudah ada
- Banyak tutorial di YouTube

**Kekurangan:**
- Tetap perlu install aplikasi di komputer
- Fitur gratis terbatas
- Lebih cocok kalau kamu sudah sedikit paham coding

**Download:** [cursor.com](https://cursor.com)

## Bolt.new — Langsung Coba di Browser, Tanpa Install

Bolt.new adalah website tempat kamu bisa ngetik deskripsi aplikasi, dan langsung lihat hasilnya di browser yang sama. **Tidak perlu install apapun.**

**Kelebihan:**
- **Zero setup** — buka website, langsung bisa mulai
- Langsung lihat hasilnya secara visual
- Cocok untuk prototyping cepat
- Bisa share link ke teman

**Kekurangan:**
- Fitur terbatas untuk proyek besar
- Tidak bisa akses file di komputermu
- Versi gratisnya cukup terbatas

**Coba sekarang:** [bolt.new](https://bolt.new)

## Replit — Workspace Coding Lengkap di Browser

Replit adalah platform coding online yang sudah include semuanya: editor, terminal, hosting, dan AI assistant.

**Kelebihan:**
- Semua di browser — install nol
- Bisa collaborate dengan orang lain secara real-time
- Ada AI built-in yang bantu nulis kode
- Langsung bisa deploy (bikin online)

**Kekurangan:**
- Versi gratis terbatas
- Koneksi internet harus stabil
- Untuk proyek besar bisa terasa lambat

**Coba sekarang:** [replit.com](https://replit.com)

## Rekomendasi Berdasarkan Skenario

### "Saya belum pernah coding dan agak takut"
Mulai dengan **Bolt.new**. Buka website, ketik "buat landing page dengan background biru dan tombol hijau", lihat hasilnya langsung. Zero pressure.

### "Saya mau belajar serius"
Mulai dengan **Cursor**. Install, buka tutorial, dan mulai dengan proyek sederhana. AI-nya akan bantu kamu setiap langkah.

### "Saya mau bikin proyek beneran"
**Claude Code** adalah pilihan paling powerful. Butuh setup sedikit lebih banyak, tapi hasilnya sepadan.

### "Saya mau coba yang paling gampang dulu"
**Replit** — semuanya online, gratis, dan ada AI-nya. Cukup daftar dan mulai.

## Checklist Sebelum Mulai

Apapun tool yang kamu pilih, pastikan:

- [ ] Komputer/laptop dengan koneksi internet
- [ ] Email untuk daftar akun (gratis)
- [ ] 30 menit waktu tanpa gangguan untuk tutorial pertama
- [ ] Satu ide sederhana yang ingin kamu buat

> **Ingat:** Tool terbaik adalah tool yang KAMU pakai. Jangan habiskan waktu terlalu lama memilih.
> Pilih satu, coba buat sesuatu, dan ganti kalau tidak cocok.
